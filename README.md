# AI Resume Analyzer

## Overview

AI Resume Analyzer is a web application that helps users analyze their resumes and receive feedback to improve ATS (Applicant Tracking System) compatibility. It identifies missing skills, provides suggestions, and helps users create stronger resumes.

## Features

* Resume upload
* ATS score analysis
* Skill extraction
* Resume improvement suggestions
* User-friendly interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Machine Learning

## Installation

1. Clone the repository.
2. Install the required packages:

   ```
   pip install -r requirements.txt
   ```
3. Run the application:

   ```
   python app.py
   ```
